# How to run this project
1. First make sure you have Node.js installed.
To verify run the command ``` node --version ```

2. If you do not have node.js installed go to: https://nodejs.org/en/ 

3. After you have node installed, clone the project into your workspace

4. Once clones, run ``` npm install ```, this command will ensure all neccessary libraries are installed to run the project

5. Finally run ``` ng serve --open ```, this command will run the project
